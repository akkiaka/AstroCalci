# 🌌 AstroCalci - Celestial Vector Calculator

A beautifully themed, functional Android application designed to handle astrological math profiles. It maps complex operations down into automated programmatic loops to replace legacy spreadsheet dependency.

## ✨ Features
* **Global Configuration Header:** Input master variables (Divide, Addition, Multiply) *once* to instantly update all active computations.
* **24 Unique Celestial Rows:** Scaled sequentially with explicit glyphs matching the core 12 planetary nodes and 12 structural zodiac vectors.
* **Real-time Cumulative Compilation:** Keeps track of active data entries dynamically to display an exact, formatted Grand Total.
* **Glassmorphic Nebula Aesthetic:** A high-contrast deep space palette using starry gold accents and frosted glass data fields.

## 🛠️ Built With
* Modern Native Android Architecture
* **Kotlin** - Language core logic and operations
* **XML Layouts** - Custom Scrollable Material Design hierarchy
* **BigDecimal Framework** - To prevent float rounding drift and retain absolute mathematical parity with precision instruments.
